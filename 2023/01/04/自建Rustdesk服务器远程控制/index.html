# 前言

我对远程控制一直有着莫名的执念，在之前陆陆续续尝试了向日葵、Todesk 等国内远程控制软件后，被安利了 Rustdesk 这个开源软件，界面、外观深得我心，并且也非常的轻量，在后台的占用很少，并且最大的特点就是 -- 不限速，其他的各类软件都需要各种 vip，实在是难以接受；并且它作为一款开源软件，它的一切都是在掌握之中的，保证过程的安全快速，因此 Rustdesk 成了我远程控制的最好选择。


但是由于 rustdesk 是一款国外的软件，他提供的免费服务器是架设在国外的，对于我们国内用户来说会有些影响速度，但官方给出了自建服务器的方法，就可以用自己的服务器很好的解决这个问题。

# 前期准备

  • 一台有公网 ip 的服务器
  • 安装 docker、docker-compose
